21.12.2011 Инвестиционный совет при губернаторе Нижегородской области одобрил предоставление земельного участка для реализации этого проекта.

ООО ТД "Новые строительные технологии" до 2016 года инвестирует 90 миллионов рублей в строительство завода по производству строительных блоков из фибропенобетона в Нижнем Новгороде, сообщил журналистам во вторник исполнительный директор предприятия Андрей Пушкарев.

Инвестиционный совет при губернаторе Нижегородской области одобрил предоставление земельного участка для реализации этого проекта.

По словам Пушкарева, строительство завода начнется через год и продлится два года. Его мощность составит 280 кубических метров блоков в сутки, что соответствует количеству, которое можно использовать для строительства двух двухэтажных коттеджей площадью по 200 квадратных метров.

Продукция предприятия предназначена для малоэтажного и монолитно-высотного домостроения.
